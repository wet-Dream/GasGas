### 가스량
-  노즐 가스분출량(단위정리 다름)
	- $Q=0.009D^2 \times \sqrt{\frac{P}{d}}=0.011KD^2 \times \sqrt{\frac{P}{d}}$ 
	$$\begin{align*}
Q : 분출가스량(m^3/h)\\
D : 노즐지름(mm)\\
K : 유출계수\\
P : 노즐 직전 압력(mmH_2O)
\end{align*}$$
- 연소기 노즐조정 : $\frac{D_2}{D_1}=\frac{\sqrt{WI_1\sqrt{P_1}}}{\sqrt{WI_2\sqrt{P_2}}}$  (아무 압력이나 넣어도 나옴)
- 웨버지수 : $WI=\frac{H_g}{\sqrt{d}}$   $H_g : 도시가스 총발열량(kcal/m^3$ 
- 도시가스 월사용 예정량
	- $Q(m^3)=\frac{(240A)+(90B)}{11000}$
	- A : 산업용 연소기 소비량합계(**kcal/h**), B : 산업용 외
- **가스홀더** 공급가스량(활동량) #오답 요소들
	- $\Delta V=V \times \frac{P_1-P_2}{P_0}\frac{T_0}{T_1}$
$$\begin{align*}
\Delta V(Nm^3)\\
P_1 : 최고사용압력(kgf/cm^2 \cdot a)\\
P_2 : 최저사용압력\\
P_0 : 표준대기압 \\
T_{0}: 표준온도\\
T_{1}: 공급시 온도 
\end{align*}$$
- **안전밸브 분출유량** : $Q(m^3/min)=0.0278PV(Mpa \cdot L)$

- **안전밸브 분출면적**

	- a : 분출부 유효면적($cm^2$)
	- W : 분출가스량(kg/h)
	- P : 분출압력($kgf/cm^2 \cdot a$)(분출압력이니까 **절대**)
	- M : 가스분자량
	- 분출직전 가스온도(K)

	- L: 압력용기 길이(mm)

			- $V_H$ : 최종단 or 최종원 *기통의(고온측) 압출량($m^3/h$)
			- $V_L$ : 최종단 or 최종원 앞의 기통의 압출량($m^3/h$)
		- *회전식 : $V=60\times 0.785tn(D^2-d^2)$ 
			- t : *회전피스톤의 가스 압축부분 두께(m)
			- n : 피스톤 회전수(rpm)
			- D : 기통안지름(m)
			- d : 피스톤 바깥지름(m)
		- *왕복동형* : $V=60\times 0.785D^2LNn$
			- D : 기통 **안지름(m)
			- L : **피스톤 행정길이(m)**
			- N : **실린더 수**(왕복동형만)
			- n : 회전수
		- *나사식* : $V=60\times C_vD^2Ln$ 
			- D : **암로터**의 지름
			- L : **로터길이**
			- n : 숫로터 회전수
			- C : **로터모양상수**
- ### 펌프관련

	- 입상배관 압력손실 : $\Delta P=1.293(d-1)h$ (1.293은 공기밀도)
		- h(m) : 입상높이, $\Delta P=mmH_2O$ **
	- $P_T=\frac{\gamma QH_f}{\eta}=\frac{\Delta PQ}{\eta}$
