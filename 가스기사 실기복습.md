#### 보일러
- 도시가스형 **반 밀폐식 보일러**설치시 **자연배기식 단독배기통 방식**의 배기통 높이 /
	- $h=\frac{0.5+0.4n+0.1l}{(\frac{1000A_v}{6H})^2}$ 
	- h : 배기통 높이(m)
	- n : 배기통 굴곡수
	- A : 배기통 유효단면적($cm^2$)
	- H : 가스소비량(kcal/h)
	- l : 역풍방지장치 개구부 하단부터 배기통 끝 개구부까지의 전길이(m)
- 공동, 반밀폐식, 강제배기식 가스보일러를 설치할때 연돌의 유효단면적 인자 / 
	- $A=H \times 0.6 \times KF + P$ 
	- H : **보일러 가스소비량 합계**(kcal/h)
	- K : 형상계수
	- F : 보일러의 **동시사용률**
	- P : **배기통의 수평투영면적**($mm^2$) 
- 개방형, 반밀폐형, 밀페형 연소장치의 연소용공기를 취하고 연소폐가스를 처리하는 방법을 각 설명하라 /
	- 개방형 : 실내에서 연소용 공기를 취하고 실내에서 연소폐가스를 방출한다
	- *반밀폐형* : 실내에서 연소용 공기를 취하고 실외에서 *연소기 배기통*을 이용하여 폐가스 배출
		- 자연배기식(*CF* : Conventional Flue) : 연소용 공기는 가스보일러가 설치되어있는 실내에서 취하고, 연소폐가스는 **자연통풍**을 이용하여 외부로 방출
		- 강제배기식(*FE* : Forced Exhaust) : 연소용공기는 가스보일러가 설치되어있는 실내에서 취하고, 연소폐가스는 **배기팬**을 이용해 **배기통**으로 외부로 방출
	- *밀폐형* : 실외에서 2단구조 배기통을 이용하여 연소용공기를 취하고 실외에서 연소기 배기통을 이용하여 폐가스 배출
		- 자연급배기식(BF)
		- 강제급배기식(FF : Forced ~ Flue) : 연소용 공기는 가스보일러가 설치된 곳의 실외에서 취하고, 연소폐가스는 **배기팬**을 이용해 **배기통**으로 외부로 방출
- 주거용 가스 보일러 종류 중 3가지 쓰고 설명
	- 단독$\cdot$밀폐식$\cdot$강제급배기식
	- 단독$\cdot$반밀폐식$\cdot$강제배기식
	- 공동$\cdot$반밀폐식$\cdot$강제배기식

	- **음속보다도** 화염 전파속도가 큰 경우로서, **가스의 경우** 1000~5000m/s에 달하여 파면선단에 충격파가 생격 **격렬한 파괴작용**을 일으키는 현상

		- DID가 짧아지는 조건
			- *정상연소속도*가 큼
			- 관속에 *방해물*이 있거나 *지름*이 작다
			- *압력*이 높다
			- *점화원의 에너지*가 크다

	- *구조가 복잡하고 가격이 비싸다*
	- *압력조정기*를 설치하여야 한다

	- **타 연료와 비교**하여 발열량이 크다

	- *연소 시 공기량*이 많이 필요하다

	- 불꽃이 염공에 접하여 연소하지 않고 염공을 떠나 공간에서 연소하는 현상
	- 분출속도가 연소속도보다 클 때 발생한다
- 리프팅(선화)의 발생원인

	- 1차 공기량이 과다할 때
	- 2차 공기량이 부족할 때
- 분젠식 연소장치의 특징
	- 불꽃은 내염과 외염을 형성
	- 연소속도가 크고, 불꽃길이 짧다
	- 연소온도가 높고, 연소실이 작아도 된다
	- 선화현상이 발생하기 쉽다
- 역화(back fire)
	- **정의**
		- 가스의 연소속도가 염공의 가스 분출속도보다 크게 됐을 때, 불꽃이 버너 내부에 침입하여 노즐의 선단에서 연소
	- 원인
		- 염공이 크게 되었을 때
		- **가스의 공급압력**이 낮을 때
		- 노즐의 구멍이 너무 크게 된 경우
		- 1차 공기량이 부족
		- 2차 공기량 과다
- 황염(Yellow tip) 
	- 정의
		- 불꽃의 끝이 적황색으로 되어 연소하는 현상
	- 원인
		- 연소반응이 충분한 속도로 진행x(작업형)
		- 1차 공기량 부족
		- 불꽃이 저온의 물체에 접촉
		- 연소**기구** 프레임이 냉각

	- g : 수용가 가스기구 사용량 총합계($m^3/h$)

	- K : 동시사용률(%)
		- $Q=g \times Y=g \times K$
